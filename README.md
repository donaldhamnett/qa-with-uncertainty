# Question Answering with Uncertainty

## Instructions

To run, please do the following:
1. clone and install [HuggingFace Transformers](https://github.com/huggingface/transformers.git) from source.

2. Copy our `project-scripts` directory into the repository root `transformers/`

3. Copy our `project_modules.py` into `transformers/examples/project_modules.py`

4. Replace `transformer/examples/run_squad.py` with our `run_squad.py`.

From here, you should be able to run the various experiment scripts within `project-scripts/`
